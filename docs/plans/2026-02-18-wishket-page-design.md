# 위시켓 전용 페이지 디자인

## 목적

위시켓 프로젝트 지원 시 포트폴리오 링크로 첨부할 전용 페이지.
기존 `/hire`는 일반 외주/알바용, `/wishket`은 위시켓 클라이언트 심리에 최적화.

## 핵심 전략

- 클라이언트는 10~30초 안에 판단함
- 화려함보다 **정보 밀도와 신뢰감**
- 기존 다크 테마 유지, 애니메이션 최소화
- 정보 우선, 장식 최소

## 섹션 구성

### 1. WishketHero — 3초 안에 핵심 전달

- 프로필 사진 + 이름 + 포지션
- 한 줄 요약: "현장 7년 + 개발 4년, 기획부터 배포까지 혼자 다 합니다"
- 핵심 지표 3개 가로: `프로젝트 11건` / `평균 작업 2~3주` / `기획~배포 1인 완결`
- CTA: "견적 문의하기" 버튼 → #contact 앵커

### 2. WhyMe — 왜 나를 골라야 하는가

카드 3장:
- **현장을 아는 개발자** — 화학공학 7년 현장 경험, 비즈니스 맥락 이해, 소통 비용 절감
- **1인 풀스택** — 기획/디자인/프론트/배포 전부 가능, 커뮤니케이션 비용 최소화
- **AI 연동 가능** — 단순 웹사이트를 넘어 AI 챗봇, 자동화 시스템까지 확장 가능

### 3. PricingTable — 견적 범위표

| 서비스 | 견적 범위 | 기간 |
|--------|----------|------|
| 회사소개 홈페이지 | 200~400만원 | 2~3주 |
| 랜딩페이지 | 100~250만원 | 1~2주 |
| 쇼핑몰 | 400~800만원 | 4~6주 |
| 관리자 페이지 | 300~600만원 | 3~5주 |
| AI 챗봇 연동 | 150~400만원 | 2~4주 |
| 기존 사이트 수정 | 50~200만원 | 1~2주 |

하단: "프로젝트 규모에 따라 달라질 수 있습니다"

### 4. CompactPortfolio — 심플 포트폴리오 그리드

- 3열 그리드, 카드: 썸네일 + 프로젝트명 + 카테고리 + 클라이언트명
- 클릭 시 모달 (기존 PortfolioGallery 모달 로직 재활용)
- 데이터: 기존 content/hire/portfolio/*.md 그대로 사용

### 5. WishketProcess — 작업 프로세스 (클라이언트 관점)

기존 ProcessFlow 데이터 재활용, 설명을 클라이언트 입장으로 재작성.
각 단계에서 클라이언트가 할 것 / 개발자가 할 것 명확 구분.

### 6. FAQ — 자주 묻는 질문

- 수정은 몇 번까지 가능한가요?
- 소스코드 인수인계 되나요?
- 유지보수는 어떻게 되나요?
- 디자인도 해주시나요?
- 호스팅/도메인은 어떻게 하나요?
- 작업 기간은 얼마나 걸리나요?

### 7. WishketCTA — 연락처 + CTA

- 전화, 이메일, 카카오톡
- "무료 상담받기" 버튼

## 톤 & 디자인

- 기존 다크 테마 유지 (사이트 통일감)
- 애니메이션 최소화 — ScrollReveal 정도만
- 3D mockup, parallax 등 화려한 효과 없음
- 기존 @theme 색상 토큰 그대로 사용
