## 설명

## 하이라이트
- 멀티 테넌트 아키텍처 설계
- API 기반 데이터 스위칭 자동화
- 실시간 마이그레이션 진행률 모니터링

## 임팩트
서비스 전환 시 수동 작업을 자동화하여 전환 시간을 대폭 단축했습니다.
